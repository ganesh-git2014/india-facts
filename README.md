# India Facts

The code that powers http://india-facts.in

## Dependencies

* react
* react-icons
* react-highcharts
* react-router
* styled-components

## Development

Clone the code and install the dependencies. Would recommend using `yarn` and `npm` would work fine too!

```
$ git clone https://github.com/rcdexta/india-trends
$ cd india-trends
$ yarn install 
$ yarn start
```
